# Quant Practice

Implementations of quantitative finance models and derivatives pricing methods, based on standard references such as Hull, Shreve and related texts.

## About

This repository contains a collection of small projects and experiments focused on:

- Derivatives pricing (forwards, futures, options)
- Arbitrage-free valuation
- Stochastic processes (Brownian motion, Itô calculus)
- Binomial and Monte Carlo methods
- Black–Scholes model and Greeks
- Volatility modeling

The main goal is to build practical intuition by implementing the models from scratch.

## Structure

Each folder corresponds to a specific topic or chapter, for example:

01_derivatives_basics/  
02_black_scholes/  


Inside each folder you will find:
- Python implementations
- Jupyter notebooks (when visualization is useful)
- A short README explaining the model

## References

Main references used:

- John C. Hull – Introduction to Futures and Options Markets  
- Steven Shreve – Stochastic Calculus for Finance  
- Paul Glasserman – Monte Carlo Methods in Financial Engineering
